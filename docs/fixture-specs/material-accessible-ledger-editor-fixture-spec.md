# Fixture Spec: Material Accessible Ledger and Editor

_Last updated: 2026-07-14_

## Contract covered

- `docs/workflows/material-accessible-ledger-editor-workflow-contract.md`

| ID | Scenario | Expected result |
|---|---|---|
| UX-001 | Open editor by keyboard | Focus enters title/first required field and remains contained |
| UX-002 | Close editor | Focus returns to invoking row/action |
| UX-003 | Reduced motion | Non-essential transform animations disabled |
| UX-004 | Issue row in either theme | Text and indicator remain readable without colour alone |
| UX-005 | 200% zoom | No action or field is lost/obscured |
| UX-006 | Save toast | Status announced once without focus move |
| UX-007 | Empty and filtered ledger | State and reset action remain explicit |

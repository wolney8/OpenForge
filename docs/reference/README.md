# Reference Documents

Use `docs/reference/` for safe reference material derived from the workbook or source pack, such as:

- redacted summaries
- field inventories
- workflow notes
- formula references
- approved extracts

UI and interaction references:

- `ui-iconography-protocols.md`: WCHE-style icon semantics, sizing/alignment, destructive-action behavior, and sportsbook-specific icon rules.

Do not commit sensitive raw workbook data, raw exports, screenshots, or personal information here unless explicitly approved.
